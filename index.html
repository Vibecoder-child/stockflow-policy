개인정보처리방침 (Privacy Policy)

본 방침은 StockFlow(이하 '앱')의 이용자 개인정보 보호를 위해 제정되었습니다.

1. 수집하는 개인정보 항목

본 앱은 이용자의 어떠한 개인정보(이름, 이메일, 전화번호, 기기 식별자 등)도 수집하거나 저장하지 않습니다. 모든 서비스는 별도의 회원가입 없이 익명으로 이용 가능합니다.

2. 데이터 사용 및 외부 API 연동

앱은 실시간 시장 데이터 및 뉴스 제공을 위해 아래와 같은 외부 서비스를 이용하며, 이 과정에서 개인정보는 전송되지 않습니다.

  • Finnhub API, Yahoo Finance: 주식 시세 데이터 조회
  • Naver Search API: 실시간 뉴스 검색
  • Google Gemini AI: 뉴스 분석 및 요약

3. 데이터 저장 및 보안

이용자가 설정한 관심 종목 리스트는 서버가 아닌 이용자의 모바일 기기 내부(Local Storage)에만 저장됩니다. 앱을 삭제할 경우 해당 데이터도 함께 삭제됩니다.

4. 개인정보 제3자 제공

본 앱은 개인정보를 수집하지 않으므로 제3자에게 제공하거나 공유하지 않습니다.
